# Finebills

This software were built as a test, using Ruby on Rails.
Follow these steps to install:

1. Clone Finebills repository:
$ git clone https://github.com/AnnaNee/FineBills.git

2. Requirements:
   * Ruby 2.3.1
   * Rails 5.0.0.1

3. Run bundle install:
$ bundle install

4. Migrate:
$ rake db:migrate

5. Start rails server:
$ rails s
